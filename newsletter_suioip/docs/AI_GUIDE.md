# Guide IA

Toute IA doit respecter:

-   Laravel 12
-   Vue 3 + TypeScript
-   PostgreSQL
-   API-first
-   PSR-12
-   Pas de logique métier dans les contrôleurs.

## Structure d'un module

-   Migration
-   Model
-   Policy
-   FormRequest
-   Controller
-   Service
-   Resource
-   Tests
-   Documentation.

Les livrables doivent être complets et directement exécutables.
